# 処理の大まかな流れ
ゲームが始まると新たなGameモデルが作成される。進行中画面にリダイレクトする。（progressコントローラー）\
現在のGameモデルと関連のあるプログレスモデルから、質問_idを抽出する。 \
抽出した質問_idを含まない、質問を全て呼び出し、シャッフルして1つ取り出す。
この質問を画面に表示する。 \
質問への答えを現在のGameモデルと関連付けて、プログレスモデルが保存する。 \
プログレスコントローラー内で、アルゴリズムオブジェクトを作成し、予想解答を絞っていく。 \
予想解答が0になれば、ギブアップ画面に遷移する。 \
予想解答が1になれば、問題に解答する画面に遷移する。 \
予想解答が2以上なら、次の質問を表示する。
